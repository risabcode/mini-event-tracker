# Mini Event Tracker

A web app where users can create an account, log in, and manage their events. Users can view upcoming or past events and optionally share events publicly.

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Database:** PostgreSQL (with Prisma ORM)  
- **Authentication:** JWT (JSON Web Tokens)  
- **Deployment:** Frontend on Vercel, Backend on Railway, PostgreSQL on Railway / ElephantSQL  

### Why this stack?
- **React:** Component-based, responsive UI  
- **Express:** Lightweight REST API  
- **PostgreSQL + Prisma:** Relational database with type-safe queries  
- **JWT:** Stateless authentication for easy deployment  

---

## Features

- **User Authentication:** Sign up, log in, hashed passwords, JWT auth  
- **Event Management:** Create, view, filter (upcoming/past), optional public shareable link  
- **Responsive UI:** Mobile and desktop friendly
